⚠️ Redundant Subquery Pattern
SELECT * FROM products 
WHERE id IN (
  SELECT product_id 
  FROM order_items 
  WHERE qty > 10
);

Triggers dependent nested loops on table scanning. Latency: 3.84 seconds

✓ Optimized INNER JOIN Pattern
SELECT DISTINCT p.* 
FROM products p 
INNER JOIN order_items o 
  ON p.id = o.product_id 
WHERE o.qty > 10;

Leverages primary index hashes instantly. Latency: 0.002 seconds
